The input videos are in folder 'videos'.

The output videos are in folder 'output'

The python file implementing the YOLO v4 algorithm for the videos is yolo_video.py

The weights and the cfg files for yolo are also included.

******************************************************************************************************************************'
For running the python file on a video.
Run the following command in terminal - 
python yolo_video.py --input <Path to the video> --output <Path for saving the output file> --yolo <folder containing yolo weights and cfg file> 
For eg.
python yolo_video.py --input videos/video5-1min.mp4 --output output/video5-1min.avi --yolo yolo-coco
